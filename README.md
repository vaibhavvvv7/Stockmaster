# Stockmaster (AI Company Analyzer)

Stockmaster is an AI-powered company analysis tool that leverages Google's Gemini API to provide comprehensive insights into any publicly traded company or organization. 

With a minimalist UI and dynamic data fetching, Stockmaster generates multifaceted reports by adopting four distinct AI persona lenses:

1. **Corporate Historian**: Provides the company's background, founding history, and core business overview.
2. **Financial Analyst**: Summarizes recent financial metrics, revenue, and profit statements.
3. **Corporate Strategist**: Highlights recent major news, strategic initiatives, and future projects.
4. **Stock Market Expert**: Recommends logical "buy" and "sell" price targets based on current momentum and value analysis.

## Features

- **Multi-Agent Insight Generation**: Asks the LLM four focused questions simultaneously, returned gracefully into interactive grid cards.
- **Responsive Design**: Modern, card-based grid layout styled with CSS (`stock.css`).
- **Loading States**: Real-time loading indicator to ensure good user experience during API fetches.
- **Rate-Limit Safe**: Built-in staggered API fetching to prevent 429 Errors from the Gemini API.

## Usage

1. Clone the repository.
2. Open `stockai.html` (the main file) in your browser.
3. Enter a company name into the search bar and press `Enter`.
4. Wait for the designated "agents" to complete their analysis and review the generated insights.

*Note: You will need to bring your own Google Gemini API key and replace the `API_KEY` placeholder in `stockai.html` if the current one expires.*

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Google Gemini API (`gemini-2.5-flash`)
